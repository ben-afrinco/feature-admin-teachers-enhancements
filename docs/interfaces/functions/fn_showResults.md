# Function: results

## Path
`app/Http/Controllers/LandingController.php`

## Description
Calculates and displays the final assessment results for a student.

## How it works in code
- Aggregates scores from all four skill tests (Reading, Listening, Writing, Speaking).
- Determines CEFR level.

## How to use from Browser
- Automatically shown after a student completes the full test sequence.
