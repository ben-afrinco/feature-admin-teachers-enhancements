# Function: sendMessage

## Path
`app/Http/Controllers/ChatbotController.php`

## Description
Sends a message to the AI chatbot and returns the response.

## How it works in code
- Communicates with an AI provider (e.g., Google Gemini or Ollama).
- Appends interaction to user chat history.

## How to use from Browser
- Type a message in the Chatbot modal and press enter/send.
