# Function: editUser (Admin)

## Path
`app/Http/Controllers/LandingController.php`

## Description
Updates the profile information of an existing user.

## How it works in code
- Submits changes to full name, email, or role for a specific user ID.
- Handles password updates if a new one is provided.

## How to use from Browser
- Admins click the "Edit" button next to a user in the Admin Dashboard.
