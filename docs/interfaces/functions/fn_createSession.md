# Function: store (OnlineSession)

## Path
`app/Http/Controllers/OnlineSessionController.php`

## Description
Creates a new live online session using Jitsi.

## How it works in code
- Stores session details (title, meeting link) in the database.
- Links session to a teacher and class.

## How to use from Browser
- Teachers use this via the "Create Session" button in their dashboard.
