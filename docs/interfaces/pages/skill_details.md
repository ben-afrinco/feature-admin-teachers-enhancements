# Page: Skill Details

## Path
`resources/views/student/skill_details.blade.php`

## Description
A generic template used to display results and breakdowns for any specific skill (Reading, Listening, etc).

## How it works in code
- Dynamically populated by the `ResultsController` based on the skill parameter.

## How to use from Browser
- Automatically navigated when clicking on a specific skill card in the results dashboard.
