# Page: Reading Test (Question 1)

## Path
`resources/views/student/tests/reading/q1.blade.php`

## Description
The first question of the reading proficiency assessment.

## How it works in code
- Logic in the route `/test/reading/q1` determines if it should show this static page or a dynamic variant.
- Submits to `TestController@submitReading`.

## How to use from Browser
- Read the text block.
- Select the best answer from the multiple-choice options.
- Click "Continue".
