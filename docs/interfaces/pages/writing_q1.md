# Page: Writing Test (Question 1)

## Path
`resources/views/student/tests/writing/q1.blade.php`

## Description
The open-ended writing assessment page.

## How it works in code
- Route: `/test/writing/q1`.
- Submits content to `TestController@submitWriting`.

## How to use from Browser
- Read the essay/paragraph prompt.
- Type your response in the provided text area.
- Ensure you meet the minimum word count (if applicable).
- Click "Save Response".
