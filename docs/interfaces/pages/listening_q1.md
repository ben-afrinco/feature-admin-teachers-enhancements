# Page: Listening Test (Question 1)

## Path
`resources/views/student/tests/listening/q1.blade.php`

## Description
The first question of the listening proficiency assessment.

## How it works in code
- Route: `/test/listening/q1`.
- Includes an audio player for the student to hear the prompt.

## How to use from Browser
- Click the play button on the audio player.
- Listen to the spoken English prompt.
- Select the matching textual answer.
- Click "Submit".
