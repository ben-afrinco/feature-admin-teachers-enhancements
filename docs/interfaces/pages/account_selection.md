# Page: Account Selection

## Path
`resources/views/auth/info_account.blade.php`

## Description
Page where users select their role (Student, Teacher, Admin).

## How it works in code
- Handled by `LandingController@info_account`.
- Route: `/account-selection`.

## How to use from Browser
- Navigate to the page after landing.
- Select a role and click "Next".
