# Page: Student Info Account

## Path
`resources/views/auth/info_account_students.blade.php`

## Description
Form for students to enter their personal information before starting tests.

## How it works in code
- Static view returned by the `/info_accountStudents` route.
- Submits to `account.store`.

## How to use from Browser
- Select "Student" role and proceed.
- Fill in the name, email, and phone.
