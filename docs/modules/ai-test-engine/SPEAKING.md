# AI Speaking Test Detail

The Speaking module uses a hybrid approach of browser-based transcription and phonetic distance calculation.

## 🎙 Component Overview
- **View**: `resources/views/dynamicSpeaking.blade.php`.
- **API**: Web Speech API (Chrome/Safari recommended).
- **Processing**: Client-side (JavaScript) accuracy calculation.

## 🔄 Accuracy Calculation
1.  **Target Text**: The passage generated by AI.
2.  **Student Speech**: Transcribed in real-time.
3.  **Comparison**: The two strings are compared using the Levenshtein Distance algorithm to calculate a percentage overlap.
4.  **Normalization**: Minimal noise (punctuation, case) is stripped before comparison.

## 📊 Result Handling
Unlike Writing, Speaking results are calculated on the client to provide **instant** feedback. The final accuracy score is then POSTed to `TestController@submitSpeaking` for storage.
