# Features & Core Modules

LingoPulse is structured into several high-level modules that handle the academic and administrative lifecycle.

## 🚀 Key Modules

### 1. [AI Test Engine](ai-test-engine/OVERVIEW.md)
Dynamic English proficiency assessment using Gemini, Groq, and OpenRouter.

### 2. [Assignments & Submissions](assignments/OVERVIEW.md)
Traditional teacher-led assignment flow with versioned student uploads.

### 3. [Online Sessions](online-sessions/OVERVIEW.md)
Live, real-time classrooms powered by integrated Jitsi Meet.

### 4. [Dashboards](../modules/DASHBOARDS.md)
Role-specific command centers for Admin, Teachers, and Students.
