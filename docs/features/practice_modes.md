# Practice & Strengthening Interfaces

## Overview
Non-graded practice environments for students to improve specific English skills without the pressure of a formal exam.

## UI Components
- **Practice Selection**: Grid of Reading, Listening, Writing, Speaking.
- **Task Cards**: Interactive exercises.

## Technical Implementation
- **Routes**: `/practice/readings`, `/practice/listenings`, `/practice/writings`, `/practice/speakings`.
- **View Mapping**: Directly maps to Blade views in `student.practice` directory.

## How to use from Browser
1. Home -> strengthening.
2. Select a skill area.
3. Interact with the provided exercises and submit for immediate self-correction.
